**********Project Purpose**********
-Creating a simple application for communication between multiple users.
-Educational purpose: familiarization and practice with React, React Native, MongoDB, WebSockets, Node.js, and Express.
-Understanding how client-server communication works in real-time using Socket.IO.
-Exploring database integration and basic message persistence for chat applications.

**********Project Description**********
-This project is a web and mobile chat application developed as part of the Operating Systems for Mobile Platforms laboratory.
-It allows multiple users to communicate in real time through a simple and intuitive interface.
-The main goal is to provide hands-on experience with frontend frameworks (React & React Native) and backend technologies (Node.js + Express) while using MongoDB for message storage and WebSockets for real-time communication.
